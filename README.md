# samdoidge.com

[samdoidge.com](http://samdoidge.com) running on [Jekyll](http://jekyllrb.com) with [Poole](https://github.com/poole/poole)

### Setup

Install Jekyll, clone this repo, run `jekyll serve` in terminal from the cloned folder :)
